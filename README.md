# 后端

技术栈： springboot redis

实现功能：

1. 通过试卷id查找试卷，返回Paper类
2. 上传试卷，上传Paper类
3. 上传anwser类

#### 目前实现的功能

- 可以在前端创建一张试卷，传到后端
- 前端通过id查找试卷显示在前端，
- 前端填写试卷
